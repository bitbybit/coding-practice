<p>Given a string, determine whether the number of vowels in the first half of the string is equal to the number of vowels in the second half.</p>
<ul>
<li>The string can contain any characters.</li>
<li>The letters <code>a</code>, <code>e</code>, <code>i</code>, <code>o</code>, and <code>u</code>, in either uppercase or lowercase, are considered vowels.</li>
<li>If there's an odd number of characters in the string, ignore the center character.</li>
</ul>
